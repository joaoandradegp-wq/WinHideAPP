# WinHideAPP
Learn how to hide running applications on your computer to maintain your privacy while using it.

Regardless of the reason, at some point you may want to hide running applications and tasks on your computer from curious eyes around you. Natively, Windows does not provide any built-in tool for this; however, a small and simple application can handle the job.

The program we are talking about is WinHideApp, an open-source software that allows you to hide any task or application with just a few clicks. Learn now how to perform this action quickly.
